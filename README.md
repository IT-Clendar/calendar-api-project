# 📅 Google Calendar API 연동 프로젝트

이 프로젝트는 사용자의 구글 캘린더에서 다가오는 일정을 가져와 출력하는 Python 프로그램입니다.  
Google OAuth 인증을 거쳐 사용자의 기본 캘린더에 접근합니다.

---

## 🔧 사용 기술

- Python 3.x
- Google Calendar API
- Google OAuth 2.0
- Git, GitHub

---

## ▶️ 실행 방법

```bash
# 1. 필요한 라이브러리 설치
pip install -r requirements.txt

# 2. 프로그램 실행
python main.py
